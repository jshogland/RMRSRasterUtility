Logical Analysis

This form is used to perform logical analyses given two raster datasets or a raster dataset and a constant. Types of analyses include; greater than, less than, greater than equal to, less than equal to, equal to, AND, and OR. Every band within the raster datasets will be queried for the logical operation. The outputs from the analysis consist of a raster dataset with each cell within a band having a value of either 0 or 1. A value of 1 identifies cells meeting the specified criteria while values of 0 denote the opposite. For AND and OR operators, raster cell values are assumed to have values of 1 or 0. Input raster values greater than or equal to 1 will equate to true and values less than 1 will equate to false.

Input Raster 1 dropdown and open button: Allows users to select the raster that will be used to compare against input raster or constant 2. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Operator dropdown: Allows users to select which type logical operation will be performed.
Input Raster 2 or number dropdown and open button: Allows users to select the raster that will be used to compare against input raster 1. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk or manually type in a constant.
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
