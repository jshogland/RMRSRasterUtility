Save Raster to Dataset

This form is used to save a selected raster to an new location. It can also be used to convert a function model into a conventional raster dataset.

Raster to Save dropdown and open button: Allows users to select which raster they want to save. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Output Workspace text box and open button: Allows users to select a valid output workspace to store the new raster dataset. A raster workspace must be selected before an Out Type can be selected.
Out Name text box: Allows users to specify the name of the newly created raster dataset.
Output Type dropdown: Allows users to select a valid raster dataset type. A raster workspace must be selected before an Out Type can be selected.
Save button: Saves the selected raster to a new raster dataset. Raster datasets are stored in the specified format and are named based on the Out Name text box.
