Landscape Metrics Form

This form is used to perform a Landscape Metrics analysis.

Input Raster dropdown and open button: Allows users to select the raster datasets that will be used in the Landscape Metrics analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Landscape Metric dropdown: Allows users to specify the type of metrics used in the analysis. Types include: Area, Edge, Ratio, and Region. If metric type is anything but Region then the Patch Statistics dropdown will appear.
Patch Statistics dropdown: Allows users to specify the type of patch statistic used in the analysis. Statistics include: Max, Min, Sum, Mean, Mode, Median, Variance, Standard Deviation, Unique, Entropy, and Probability (Statistics description).
Window Type dropdown: Allows users to specify the shape of the analysis window. Window Types include Rectangle and Circle. If Circle is chosen for window shape, then Columns and Rows are replaced with Radius.
Columns, Rows, and Radius numeric up down: Allows users to specify the sized of the analysis window deminsions in cells.
Out Name text box: Allows users to specify the name of the newly created function dataset.
