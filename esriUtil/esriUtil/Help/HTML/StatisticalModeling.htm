Statistical Modeling Menu

The Statistical Modeling Menu contains 5 buttons that can be used to build, design and run linear, multivariate, logistic, polytomous logistic, and accuracy assessment models. These buttons integrate SAS operating routines directly into ArcMap. SAS is required to design and run these types of models (future releases will incorporate R).
