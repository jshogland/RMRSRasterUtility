Calculate Raster Statistics

This form is used to calculate statistics for a raster given a specified skip value. It can also be used to calculate statistics for a function model.

Input Raster dropdown and open button: Allows users to select which raster will have statistics (re)calculated. The drop down contains any raster in the TOC and the open button allows the user to select any existing Function Model stored on disk (conventional raster datasets must be loaded into the TOC).
Skip Factor: Allows users to select the number of cells to skip when calculating statistics. A value of 1 means all cells are used in generating raster statistics, a value of 2 samples only the even cells, a value of 3 will sample every third cell, and etcetera.
Execute button: Starts calculating statistics for the specified raster. After clicking this button a running dialog form will appear indicating the progression of the command.
