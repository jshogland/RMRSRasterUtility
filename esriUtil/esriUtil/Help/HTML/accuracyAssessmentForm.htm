Accuracy Assessment

This form is used to build an Accuracy Assessment of a given classification. Currently this tool requires SAS to generate the accuracy assessment. Statistics calculated include chi-squared, log likelihood, kappa, and a standard error matrix probabilistic confidence intervals. For a complete description of SAS outputs please refer to the Freq and Logistic procedures.

Alpha numeric dropdown: The level of significance that will be used to build confidence intervals.
Feature Class and open button: Allows users to select the dataset that contains both mapped values and field validated values.
Mapped Field dropdown: The field used to identify the values of the mapped layer at that location.
Reference Field dropdown: The field used to specify the known (field validated) values at that location.
Weight Field dropdown: The field used to specify the weight of each sample.
Map to Update CI dropdown and open button: Allows users to select a map layer (raster surface or feature class) to update the attribute table with an error matrix of user probabilities and confidence intervals.
Edit Code checkbox: When checked the prc.sas file will be opened for editing.
Exact checkbox: When checked the Exact statement will be added to the Freq procedure of the SAS file.
Execute button: Executes the PLR routine. After clicking the Execute button a batch SAS process will open, running the prc.sas code.

Output
All outputs are located within a the workspace of the training feature class and are stored within a directory called SASOUTPUTS\ACCURACYASSESSMENT. Outputs include:

  1. A series of graphical outputs depicting the relationship between response and explanatory variables.
  2. A .csv file of the input data named after the training data feature class.
  3. A SAS log file named prc.log. This file provides all the log information produced from running the SAS procedure file.
  4. A SAS results file named prc.lst. This file provides all the results produced from running the SAS procedure file.
  5. A SAS procedure file named prc.sas. This file contains the SAS code that is submitted to SAS in the batch process. You can open this file and modify the SAS procedure to produce different outputs.
  6. A series of fields appended to the selected Map to Update CI layer's attribute table depicting users probabilities and confidence intervals associated with each cell of the error matrix. Prefix P_ denotes predicted probabilities. Prefix LCL_ denotes lower confidence limits. Prefix UCL_ denotes upper confidence limits.
