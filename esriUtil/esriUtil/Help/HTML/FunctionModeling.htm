Function Modeling is a new raster processing methodology used to create and store transformations of raster datasets. Function Modeling uses function datasets ESRI website and our open source .net code library to store all the functions occurring to a source raster and dynamically build those datasets at run time. Function modeling significantly improves raster processing a substantially reduces storage associated with raster modeling. Benefits of Function Modeling include:

  • no intermediate raster dataset creation
  • immediate depiction of raster transformations
  • faster processing
  • reduced storage
  • increased efficiency

The figure above illustrates the conceptual idea behind function modeling. For each raster transformation, conventional modeling creates an intermediate output raster dataset. Function modeling however remove this step thereby reducing processing time and storage space. Moreover, Function Modeling gives analysts the option to save models so they can be loaded into ArcMap at a later date.
