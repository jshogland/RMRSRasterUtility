Export to CSV

This form is used to export attribute tables values to a CSV file.

Sample Feature Class dropdown and open button: Allows users to select the feature class from which they want to export the attribute table information.
Output CSV text box and open button: Allows users to specify the name and location of the specified CSV file.
Export button: Saves the attribute information from the selected feature class to a new CSV file.

Output

A .csv file located at the specified location.
