Open Function Model

This form is used to to add the outputs of a Function model to the active view.

File name dropdown: Allows users to select a Function Model. Function Models are stored in a function dataset directory located within a given workspace and have a .fds extension.
Files of type: Allows users to select a function model with a .fds extension.
Open button: Opens the selected function model and adds the output (Raster) to the active view.
Cancel button: Close the open function model dialog.
