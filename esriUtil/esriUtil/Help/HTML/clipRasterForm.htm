Clip Raster

This form is used to clip a raster datasets to the boundary of a specified feature class.

Clipping Polygon dropdown and open button: Allows users to select the polygon layer that will be used to clip the specified raster. The drop down contains any polygon feature class in the TOC and the open button allows the user to select any existing polygon stored on disk.
Raster to Clip dropdown and open button: Allows users to select the raster that will be clipped.The drop down contains any raster in the TOC and the open button allows the user to select any existing polygon stored on disk.
Out Raster Name text box: Allows users to specify the name of the newly created raster dataset.
Erase check box: When checked the tool erases the area within the feature class polygon rather than erasing the area outside of polygon featrue class.
Clip button: Clips the specified raster dataset to the boundary of the specified feature class.
