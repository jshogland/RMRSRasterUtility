Help Menu

The Help menu contains 3 buttons that can be used to view documentation, the version of the RMRS Raster Utility toolbar, and the RMRS Raster Utility website. Each time ArcMap closes down the RMRS Raster Utility checks for help updates. If it finds updates it prompts the user to update their system. If the system has been updated, the user will see those updates the next time they open ArcMap.

Documentation opens the chm file associated with the RMRS Raster Utility.
Version opens a window identifying the version of the toolbar.
Opens the RMRS Raster Utility website.
