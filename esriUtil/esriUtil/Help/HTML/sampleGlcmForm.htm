Sample GLCM

This form is used to sample GLCM values given a series of sample locations. Currently some of the GLCM metrics create conventional model outputs rather than function datasets, which can substantially increase processing time. This tool allows the user to sample GLCM metrics without creating conventional dataset, providing a way to quickly extract sample values and test if those values warrant creating a conventional dataset. Future release of RMRS Raster Utility will deprecate this form and convert those metrics that produce conventional raster into function datasets. In the future users will be able to use the Create GLCM Raster Analysis tool to create a function model and then sample that newly created function model using the Sample Raster Value form.

Sample Feature Class dropdown and open button: Allows users to select the point feature class that represents the sample locations.
Raster dropdown and open button: Allows the user to select the raster that will be used to calculate GLCM surfaces.
Window Type dropdown: The type of analysis window used to look at neighboring raster values (Rectangle or Circle).
Columns, Rows, and Radius numeric dropdown: Allows users to specify the size of the analysis window. If window type is Circle the Columns and Rows numeric dropdown disapears and the Radius numeric dropdown appears. The deminsion(s) of the widow and radius are specified in cells.
Direction dropdown: Allows users to select the direction of comparison for the GLCM (horizontal or vertical).
Types dropdown: Allows users to select the types of GLCM metrics to calculate. Once selected in the dropdown the users must click on the + button to add the metric to list of GLCM metrics to sample.
+ button: Adds metric types to the list of GLCM values to sample.
- button: GLCMM values to sample.
% button: Adds all metric types to the list of GLCM values to sample.
! button: Subtracts all metrics types from the list of GLCM values to sample.
Sample button: Samples the specified GLCM metric values for the specified raster surface.


New fields within the selected sample locations feature that depict the specified GLCM values. Field names are based on the metrics selected.
