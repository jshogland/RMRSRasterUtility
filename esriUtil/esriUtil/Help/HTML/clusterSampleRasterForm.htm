Cluster Sample Raster

This form is used to sample a specified raster using the sample location of each feature within the feature class and the specified Azimuth and Distance offsets. Raster values are summarized using the Cluster Summarization Type and are added to the attribute table of the specified feature class. Field names will include the name of the raster sampled followed by the band number that was sampled.

Sample locations dropdown and open button: Allows users to select the point feature class that represents the sample locations.
Raster to Sample dropdown and open button: Allows users to select the raster that will be sampled. All bands within the specified raster will be sampled.
Cluster Summarization dropdown: Allows users to select how to summarize the clustered samples.
Sample button: Samples the specified Raster.

Output

A series of fields with summarized raster band values for the location of each sample point and that point's associated offsets.
