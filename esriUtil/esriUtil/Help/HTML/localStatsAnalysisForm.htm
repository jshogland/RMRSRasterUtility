Local Stats Analysis

This form is used to perform Local statistical analyses given specified raster datasets and statistics. Local statistics transform the overlapping cell values of each band within a raster dataset into a new statistic stored within a single band raster dataset. Statistics include: max, min, maxband, minband, sum, multiply, divide, subtract, power, mean, variance, standard deviation, mode, median, unique, and entropy. The output from the analysis consist of a raster dataset (single band) with transformed cell values.

Select Raster dropdown, open button, and listbox: Allows users to select the raster that will be used as an input to the local stats analysis. All bands within each raster dataset will be used in the analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. Once selected raster datasets will be moved to the select raster listbox. To remove a raster dataset select it within the listbox and click on the - button. To clear the select raster listbox click on the ! button.
Function: Allows users to define the type of statistic that will be calculated. These statistics are calculated in the same manner as focal statistics except "n" is determined by the number of bands as opposed to the number of cells within a focal window.
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
