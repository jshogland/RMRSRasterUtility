Extract Raster Band

This form is used to extract a raster band from a raster dataset.

Input Raster dropdown and open button: Allows users to select the raster dataset that will be used to extract a raster band. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. Check all bands you wish to extract. To remove a raster band from the check listbox selection select it from the listbox and click the - button. To clear the input listbox click the ! button.
Out Name text box: Allows users to specify the name of the newly created raster dataset.
Execute button: Performs the extract operation.
