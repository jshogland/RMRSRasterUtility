Arithmetic Analysis

This form is used to perform arithmetic analyses between 2 raster datasets or a raster dataset and a constant. Types of analyses include; plus, minus, multiply, divide, power, and modules division. If both raster datasets are multiband raster datasets with an equal number of raster bands than the tool will perform arithmetic analysis between each corresponding band of the 2 raster datasets. If one raster is a multiband raster (MR) and the other is a constant value or raster with fewer bands (FR), then a value array or multiband raster will be created (using the constant or the first band of FR) on the fly that has the same number of values or raster bands as the MR (this allows the user to perform one arithmetic function to multiple each raster band of the MR using  a single value or raster band) . There must be at least one raster dataset specified for the tool to work.

Input Raster 1 or number dropdown and open button: Allows users to select the first raster or constant in the analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. To specify a constant the user must manually type a number.
Input Raster 2 or number dropdown and open button: Allows users to select the second raster or constant in the analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. To specify a constant the user must manually type a number.
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
