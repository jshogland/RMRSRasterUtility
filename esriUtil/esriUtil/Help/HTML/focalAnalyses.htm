Focal Analysis overview

Focal analysis is concerned with summarizing neighboring cell values within an analysis window and applying that summary to a specific focal cell within the analysis window (ESRI website). The Focal cell sits in the middle of analysis window (figure 1) and can take a wide range of numeric values(figure 2). The outputs of Focal Analysis can be used as predictor variables in a model and can be combined in numerous ways using the RMRS Raster Utility.

 

Figure 1. An example of how focal analyses works. In this example the analysis window is defined by the cells with a light blue background and the focal pixel is outlined in red. All values within the analysis window are summed to calculate the focal statistic. This statistic is then applied to the focal cell of the output raster. This process is then repeated for all cells within the output dataset.
