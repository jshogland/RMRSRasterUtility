Rescale Raster

This form is used to rescale raster datasets values to the min and max numbers of the pixel types. Users can use this tool to reduce raster size while maintaining magnitude of those values. Rescaling is achieved using the ESRI stretch function min max .

Input Raster dropdown and open button: Allows users to select the raster layer to rescale. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Pixel Depth dropdown: Allows users to select the min and max numbers of the output raster (min and max numbers are derived by pixel types).
Out Name text box: Allows users to specify the name of the newly created raster dataset.
Execute button: Performs the rescale operation.
