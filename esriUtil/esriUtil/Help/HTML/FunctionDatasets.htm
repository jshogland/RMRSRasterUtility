Function Datasets Overview

Image Analysis Window

Function Datasets are a new raster dataset type created by ESRI that allow users to store transformations of a raster datasets without actually creating standard raster dataset. Within ArcMap, function datasets can be created and manipulated using the Image Analysis Window and can be visualized while the current ArcMap session is open. Closing ArcMap deletes the existing Function Datasets. For more information about Function Datasets please visit ESRI website
