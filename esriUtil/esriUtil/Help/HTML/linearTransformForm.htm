Linear Transform Analysis

This form is used to perform a linear transform (regression) of specified raster datasets. Each band of each raster dataset selected is added to the slope rasters data grid. For each band a slope value must be specified. To specify that there is no intercept for the transformation use a values of 0. To use categorical raster surfaces, convert each category within a given categorical surface to a binary raster representation (cell values of 0 or 1).  The outputs from the analysis consist of a single band raster dataset with transformed cell values based on intercept and slope parameters.

Intercept numeric up down: Allows users to specify an intercept value.
Select Slope Rasters drop down, open button, and listbox: Allows users to select rasters that represent input parameters to the linear transformation and specify a slope value for each band within each raster dataset. To remove a raster band select that band and click on the - button. To clear the listbox click on the ! button.
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
