Web Services Menu

The Web Services Menu contains 5 buttons that can be used to build a database and store data from a specified web service.
After specifying services and layers, users can zoom to an area of interest and click on the Service Download button.
For the extent of the view all feature classes and raster surfaces selected will be downloaded and stored at the specified database.
For image services, a user can also use the Tiled Image Service Download tool. This tool will create a series of image tiles based on the
amount of data that can be retrieved from a given image service request.
