Explode Samples

This form is used to explode sample locations that represent multiple plot locations (for example, FIA sampling design plot locations that use subplot).

Sample Locations dropdown and open button: Allows users to select from which feature class they want to explode the sample locations.
Link Field: Allows users to specify the name of the field used to link clustered points to the original plot center.
Offset from center data grid: Allows user to specify azimuth and distance offsets from the center plot location. Each Azimuth and Distance value will create a new point in the output feature class.
Explode button: create a new point feature class that contains the central point of the Sample locations layer along with an additional point for each Azimuth and distance specified in the Offset from center data grid. Output feature classes are stored in the same workspace as the input feature class and are named the same as the input feature class with a "_explode" suffix.

Output

A new point feature class.
