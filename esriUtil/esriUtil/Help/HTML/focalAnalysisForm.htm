Focal Analysis

This form is used to perform focal analyses given a raster dataset, specified focal window, and statistic. Types of analyses include; max, min, sum, mean, mode, median, variance, standard deviation, unique, entropy, and probability (statistical formulas). Every band within the raster dataset will be transformed based on the analysis specifications. The outputs from the analysis consist of a raster dataset with transformed cell values based on neighboring cell values and the focal statistic.

Input Raster dropdown and open button:on: Allows users to select the raster that will be used as an input to the focal analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Focal Statistic: Allows users to select the focal statistic.
Window Type dropdown: Allows users to select the type of analysis window rectangle or circle. If circle is selected than Columns and Rows are replaced with a Radius numeric up down control.
Columns: Allows users to select the number of columns (horizontal cells) for a given focal window.
Rows:ws: Allows users to select the number of rows (vertical cells) for a given focal window.
Radius:us: Allows users to select the radius size for a given focal window (cells).
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
