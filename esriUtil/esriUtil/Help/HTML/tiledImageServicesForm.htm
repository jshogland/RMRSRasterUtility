Tiled ImageServer Download

This form is used to select an image service, an area of interest, and a database to begin the download process. After download is clicked, a progress form will open specifying the status of the download. Images will be tiled based on maximum size of data requests from the server and will be named sequentially based on a row column gridding.

Image Service Layer dropdown : Allows users to select which which image service layer they want to download from the existing image service layers in the TOC.
Extent dropdown: Allows the user to select which extent they wish to download (typically the extent of the display).
Database button: Allows user to select database in which image tiles will be stored.
Download button: Begins the download process.
